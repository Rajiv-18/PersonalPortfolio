## üöÄ **Getting Started**

To run this portfolio locally:

### Prerequisites
- A web browser
- (Optional) Python 3.x or Node.js for local server

### Installation Steps

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Rajiv-18/PersonalPortfolio.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd PersonalPortfolio
   ```

3. **Open in your browser or start a local server:**

   **Using Python:**
   ```bash
   python -m http.server 8000
   ```

   **Using Node.js:**
   ```bash
   npx http-server
   ```

4. **Visit your local server:**
   ```
   http://localhost:8000
   ```
